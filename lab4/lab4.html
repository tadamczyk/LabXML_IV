Funkcja Dirichleta

Funkcja charakterystyczna zbioru liczb wymiernych Q, tzn. funkcja zmiennej rzeczywistej, która przyjmuje wartość 1, gdy argument jest liczbą wymierną i wartość 0, gdy argument jest liczbą niewymierną.
Formalnie funkcję Dirichleta można zapisać wzorem:
1 Q ⁡ x = 1 dla x∈Q 0 dla x∉Q
